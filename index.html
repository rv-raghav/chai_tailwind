Zero-build utility experiment

Brew your own utility CSS with chai-* classes.

This repo started as a simple proof of concept. Now it behaves more like a tiny utility engine: it scans chai- classes, generates CSS at runtime, and gives you a clean landing page to show what the system can do.

Utility sampler

Keep the engine simple, but give it real range.

The original version covered padding, margin, colors, and a few text helpers. This version adds layout, sizing, shadows, borders, radius, and a warmer design language so the framework feels more intentional.
